## ⚙️ Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** TailwindCSS + Custom Animations
- **Animations:** Framer Motion
- **Fonts:** Google Fonts via `next/font`
- **Deploy:** Vercel (Free & Fast)
- **Extras:** Dark Mode, SEO Optimized, 100% Responsive
